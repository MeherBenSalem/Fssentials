# Fssentials - Playful but Professional Moderation Essentials

Fssentials is a powerful and user-friendly plugin designed for Paper servers. It provides essential tools for server moderation while maintaining a playful and professional approach. Whether you're managing a small community or a large network, Fssentials has you covered.

---

## Features

### Moderation Tools
- **Punishment System**: Manage bans, mutes, and warnings with ease.
- **Vanish Mode**: Go invisible to monitor players discreetly.
- **Admin Tools**: Access essential commands for server management.

### Maintenance Utilities
- **Maintenance Mode**: Restrict server access during updates or downtime.
- **Timers and Schedulers**: Automate tasks with precision.

### Inventory Management
- **InvSee**: View and manage player inventories in real-time.

### Customization
- **Configurable Messages**: Tailor messages to fit your server's theme.
- **YAML Storage**: Lightweight and easy-to-edit configuration files.

### Compatibility
- **Paper API**: Built specifically for Paper servers, ensuring optimal performance.
- **Java 21 Support**: Fully compatible with the latest Java LTS version.

---

## Installation

1. Download the latest version of Fssentials from [CurseForge](https://www.curseforge.com) or [Modrinth](https://modrinth.com).
2. Place the downloaded `.jar` file into your server's `plugins` folder.
3. Restart your server.
4. Configure the plugin using the `config.yml` file located in the `plugins/Fssentials` directory.

---

## Commands

| Command | Description |
|---------|-------------|
| `/ban <player>` | Ban a player from the server. |
| `/mute <player>` | Mute a player. |
| `/vanish` | Toggle vanish mode. |
| `/maintenance` | Enable or disable maintenance mode. |
| `/invsee <player>` | View a player's inventory. |

---

## Permissions

| Permission Node | Description |
|-----------------|-------------|
| `fssentials.ban` | Allows banning players. |
| `fssentials.mute` | Allows muting players. |
| `fssentials.vanish` | Allows using vanish mode. |
| `fssentials.maintenance` | Allows toggling maintenance mode. |
| `fssentials.invsee` | Allows viewing player inventories. |
